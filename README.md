## VR Creation - Premium Embroidery Portfolio 🧵✨

A high-end, responsive single-page portfolio website developed for **VR Creation**, a boutique embroidery studio. This project showcases premium handcrafted needlework with a focus on luxury UI/UX and seamless Instagram integration.

## 🌟 Key Features
- **Luxury Aesthetic:** Dark-themed UI with golden accents for a premium boutique feel.
- **Interactive Gallery:** Masonry layout with "Grayscale to Color" hover effects.
- **Custom Experience:** Smooth parallax scrolling and a custom-branded mouse cursor.
- **Business Focused:** Integrated "DM on Instagram" call-to-action buttons for lead conversion.
- **Fully Responsive:** Optimized for all screen sizes (Mobile, Tablet, and Desktop).

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript.
- **Design Partner:** Developed with architectural assistance from Claude AI.
